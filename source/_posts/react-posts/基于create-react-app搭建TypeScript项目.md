# 准备工作

1.安装`node`环境

官网：`https://nodejs.org/zh-cn/`

建议安装长期支持版本


2.安装`create-react-app`

官网：`https://create-react-app.dev/`


# 创建项目

- 创建基于`TypeScript`的版本

```javascript
npx create-react-app my-app --template typescript
cd my-app
npm start
```
